Ink Pen Assembly Cell Simulation

Advanced Programming Project

Project Overview

This project simulates an ink pen production line using Python, a custom HMI, InfluxDB, Grafana, and Docker. The system models the manufacturing process of an ink pen and displays live production data in a Grafana dashboard.

Production Line Stages

  • S1: Ink cartridge loading
  • S2: Nib press-fit
  • S3: Barrel closing
  • S4: Cap and clip mount
  • S5: Write-test inspection

System Components

  • Python backend for production logic
  • Tkinter HMI for operator control
  • InfluxDB for time-series production data
  • Grafana dashboard for live monitoring
  • Docker Compose for running InfluxDB and Grafana

Dashboard Parameters

  • Produced pens
  • Accepted pens
  • Rejected pens
  • Cell temperature
  • Machine state
