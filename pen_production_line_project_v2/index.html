Ink Pen Production Line Simulation

Advanced Programming Project by Mostafa Elsayed, Student ID 100006129. This website describes a software-based manufacturing cell for assembling ink pens using Python, Tkinter HMI, InfluxDB, Grafana, and Docker.

Production Concept

The simulated product is an ink pen. The line assembles the ink cartridge, nib, barrel body, cap, and clip before a final write-test quality inspection.

HMI Design

The HMI uses a dark industrial operator-console layout with a control panel, station flow display, KPI cards, and event log.

Database and Dashboard

Production data is written to InfluxDB in the pen_line bucket and visualized in Grafana panels.

Assembly Stages

  • S1: Ink cartridge loading
  • S2: Nib press-fit
  • S3: Barrel closing
  • S4: Cap and clip mount
  • S5: Write-test inspection

Tools Used

  • Python and Tkinter for backend and HMI
  • Docker Compose for service deployment
  • InfluxDB for time-series production data
  • Grafana for live dashboard visualization
  • GitHub and GitHub Pages for hosting

Repository

The source code can be hosted in a GitHub repository and the website can be published using GitHub Pages.
